Chapter 3. The Menu Entries

The File Menu

File->New (Ctrl+N)
This command start a new document in editing window. The new document is named Untitled.

File->Open (Ctrl+O)
Launch open file dialog box to let you open one or more files.

File->Save (Ctrl+S)
Save your file. Use it often. If the file is Untitled, then Save becomes Save As.

File->Save As (Shift+Ctrl+S)
Name and rename file with this command. It launch the save file dialog box to save a file.

File->Page Setup
Launch the page setup dialog to set the page for printing.

File->Print (Ctrl+P)
Launch the print dialog to print the current file to a printer or a file.

File->Close (Ctrl+W)
Close the active file with this command, If you have made unsaved changes, you will be prompted to save the file before close it.

File->Quit (Ctrl+Q)
This command closes Jollpi and any files you were editing. If you have made unsaved changes to any of the files you were editing, you will be prompted to save them.

 The Edit Menu

Edit->Undo (Ctrl+Z)
Undo the last editing command (typing, copying, cutting, etc.

Edit->Redo (Shift+Ctrl+Z)
Redo the last undo step.

Edit->Cut (Ctrl+X)
Removes selected text if any, and places a copy of the removed text in the clipboard.

Edit->Copy (Ctrl+C)
Copies selected text, if any, to the clipboard.

Edit->Paste (Ctrl+V)
Copies the first item in the clipboard into the editor at cursor position.

Edit->Select All (Ctrl+A)
Selects all text in the editor.

Edit->Delete
Delete all selected text in the editor

 The Search Menu

Search->Find (Ctrl+F)
Launch the find dialog to allow you to search for text in the edited documents.

Search->Find Next (F3)
Go to nearest downwards match of the last text searched for, starting from cursor position.

Search->Find Previous (Shift + F3)
Go to nearest upwards match of the last text searched for, starting from cursor position.

Search->Replace (Ctrl+R)
Launch the replace dialog to replace one or more of defined text with something else.

Search->Go to Line (Ctrl+G)
Launches the go lo line Dialog, allowing you to enter the number of a line to find in the document.

 The View Menu

View->Wrap
This menu lists the available wrap text mode. Wrap text makes all the text in a view visible without the need for horizontal scrolling by rendering one actual line on more visual lines as needed.

View->Highlighting Style
This menu lists the available highlight mode .You can change the highlight for the current view here.

View->Enable Auto Indent (F8)
Toggle to enable or disable auto indent for your edited documents.

View->Show Line Numbers (F11)
Toggle to show or hide line numbers in the left border for your edited documents.

View->Show Line Marks (F9)
Toggle to show or hide line marks in the left border for your edited documents.

View->Show Right Margin (F7)
Toggle to show or hide right margin for your edited documents.

View->Select Font (F6)
Launch the font selection dialog to change familiy, style, or size from your current font.

 The Help Menu

Help->Contents (F1)
This will display Jollpi manual.

Help->About
This will display version and author information.
